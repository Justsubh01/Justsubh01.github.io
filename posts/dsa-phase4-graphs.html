Graphs: BFS, DFS, and Classic Interview Problems

Graphs are the most versatile data structure in computer science and a staple of FAANG interviews. Once you understand adjacency list representation, DFS, and BFS, you have the tools to solve connected components, shortest path, cycle detection, bipartite check, and most graph problems.

The dirty secret of graph problems: most of them are just "run BFS or DFS from some starting nodes, tracking visited nodes." The complexity comes from knowing when to use BFS vs DFS, what to track, and how to handle the specific problem constraints. Get the traversal right; the rest is logic.

Graph Representations

  Graph: G = (V, E) — vertices and edges.
  Directed: edges have direction (A → B ≠ B → A)
  Undirected: edges are bidirectional (A — B)
  Weighted: edges have weights
  Unweighted: all edges equal

  ─── Adjacency List ──────────────────────────────────────
  # Best for sparse graphs (most real-world graphs)
  graph = {
      0: [1, 2],
      1: [2],
      2: [3],
      3: []
  }
  # Or: defaultdict(list)
  from collections import defaultdict
  graph = defaultdict(list)
  graph[0].append(1)  # add edge 0 → 1

  Space: O(V + E)
  Edge lookup: O(degree(v))
  Neighbor iteration: O(degree(v))

  ─── Adjacency Matrix ────────────────────────────────────
  # Best for dense graphs or when O(1) edge lookup needed
  V = 4
  matrix = [[0]*V for _ in range(V)]
  matrix[0][1] = 1  # edge 0 → 1

  Space: O(V^2)
  Edge lookup: O(1)
  Neighbor iteration: O(V)

  ─── Edge List ───────────────────────────────────────────
  # Simplest, useful for Kruskal's MST
  edges = [(weight, u, v) for each edge]

  ─── Grid as Graph ───────────────────────────────────────
  # Extremely common in FAANG interviews
  # Each cell is a vertex, edges to 4 adjacent cells (or 8 for diagonal)
  DIRECTIONS = [(0,1),(0,-1),(1,0),(-1,0)]
  def neighbors(r, c, rows, cols):
      for dr, dc in DIRECTIONS:
          nr, nc = r+dr, c+dc
          if 0 <= nr < rows and 0 <= nc < cols:
              yield nr, nc

Depth-First Search (DFS)

  # DFS: go as deep as possible, backtrack when stuck.
  # Natural with recursion (using call stack).

  ─── Recursive DFS ───────────────────────────────────────
  def dfs(graph, node, visited):
      visited.add(node)
      for neighbor in graph[node]:
          if neighbor not in visited:
              dfs(graph, neighbor, visited)

  # Full connected components:
  def count_components(n, edges):
      graph = defaultdict(list)
      for u, v in edges:
          graph[u].append(v)
          graph[v].append(u)  # undirected

      visited = set()
      count = 0
      def dfs(node):
          visited.add(node)
          for nb in graph[node]:
              if nb not in visited:
                  dfs(nb)

      for node in range(n):
          if node not in visited:
              dfs(node)
              count += 1
      return count

  ─── Iterative DFS (using explicit stack) ────────────────
  def dfs_iterative(graph, start):
      visited = set()
      stack = [start]
      result = []
      while stack:
          node = stack.pop()
          if node in visited: continue
          visited.add(node)
          result.append(node)
          for nb in graph[node]:
              if nb not in visited:
                  stack.append(nb)
      return result

  DFS time: O(V + E) — visit each vertex and edge once
  DFS space: O(V) — recursion depth or stack

Breadth-First Search (BFS)

  # BFS: explore level by level.
  # Uses a queue. Finds SHORTEST PATH in unweighted graphs.

  from collections import deque

  def bfs(graph, start):
      visited = {start}
      queue = deque([start])
      result = []
      while queue:
          node = queue.popleft()
          result.append(node)
          for nb in graph[node]:
              if nb not in visited:
                  visited.add(nb)
                  queue.append(nb)
      return result

  ─── BFS Shortest Path (Unweighted) ──────────────────────
  def shortest_path(graph, start, end):
      if start == end: return 0
      visited = {start}
      queue = deque([(start, 0)])   # (node, distance)
      while queue:
          node, dist = queue.popleft()
          for nb in graph[node]:
              if nb == end: return dist + 1
              if nb not in visited:
                  visited.add(nb)
                  queue.append((nb, dist + 1))
      return -1  # unreachable

  BFS time: O(V + E)
  BFS space: O(V) — queue can hold at most V vertices

  BFS vs DFS:
  ─────────────────────────────────────────────────────────────
  BFS: shortest path (unweighted), level-order, finds closest
  DFS: cycle detection, topological sort, backtracking, all paths

Grid BFS/DFS: The Island Pattern

  # Islands problem: count connected components of '1's in binary matrix.
  # This template appears in dozens of variations.

  def num_islands(grid):
      rows, cols = len(grid), len(grid[0])
      visited = set()
      count = 0

      def dfs(r, c):
          if (r, c) in visited or r<0 or r>=rows or c<0 or c>=cols:
              return
          if grid[r][c] != '1':
              return
          visited.add((r, c))
          dfs(r+1, c); dfs(r-1, c); dfs(r, c+1); dfs(r, c-1)

      for r in range(rows):
          for c in range(cols):
              if grid[r][c] == '1' and (r, c) not in visited:
                  dfs(r, c)
                  count += 1
      return count

  ─── Variations on Islands ────────────────────────────────
  Max area island:     count cells in DFS, track maximum
  Surrounded regions: flood fill from borders first
  Pacific/Atlantic:    BFS/DFS from each ocean boundary
  Shortest bridge:     DFS first island, BFS to second island

Cycle Detection

  ─── Undirected Graph: DFS with Parent ───────────────────
  def has_cycle_undirected(n, edges):
      graph = defaultdict(list)
      for u, v in edges:
          graph[u].append(v)
          graph[v].append(u)

      visited = set()
      def dfs(node, parent):
          visited.add(node)
          for nb in graph[node]:
              if nb == parent: continue   # don't go back through same edge
              if nb in visited: return True  # found cycle
              if dfs(nb, node): return True
          return False

      for node in range(n):
          if node not in visited:
              if dfs(node, -1): return True
      return False

  ─── Directed Graph: DFS with Three Colors ───────────────
  # 0 = unvisited, 1 = in current path (gray), 2 = done (black)
  def has_cycle_directed(n, edges):
      graph = defaultdict(list)
      for u, v in edges:
          graph[u].append(v)
      color = [0] * n
      def dfs(node):
          color[node] = 1   # start processing
          for nb in graph[node]:
              if color[nb] == 1: return True    # back edge = cycle
              if color[nb] == 0 and dfs(nb): return True
          color[node] = 2   # done processing
          return False
      return any(dfs(i) for i in range(n) if color[i] == 0)

Bipartite Check

  # A graph is bipartite if its vertices can be divided into two sets
  # such that all edges connect vertices from different sets.
  # Equivalent: graph has no odd-length cycles.

  def is_bipartite(graph):
      n = len(graph)
      color = [-1] * n
      def bfs(start):
          queue = deque([start])
          color[start] = 0
          while queue:
              node = queue.popleft()
              for nb in graph[node]:
                  if color[nb] == -1:
                      color[nb] = 1 - color[node]  # opposite color
                      queue.append(nb)
                  elif color[nb] == color[node]:
                      return False  # same color = not bipartite
          return True

      for i in range(n):
          if color[i] == -1:
              if not bfs(i): return False
      return True

Clone Graph

  def clone_graph(node):
      if not node: return None
      cloned = {}  # old node → cloned node
      def dfs(n):
          if n in cloned: return cloned[n]
          copy = Node(n.val)
          cloned[n] = copy
          for nb in n.neighbors:
              copy.neighbors.append(dfs(nb))
          return copy
      return dfs(node)

2. Max Area of Island

  def max_area_of_island(grid):
      rows, cols = len(grid), len(grid[0])
      def dfs(r, c):
          if r<0 or r>=rows or c<0 or c>=cols or grid[r][c] != 1:
              return 0
          grid[r][c] = 0  # mark visited by setting to 0
          return 1 + dfs(r+1,c) + dfs(r-1,c) + dfs(r,c+1) + dfs(r,c-1)
      return max(dfs(r,c) for r in range(rows) for c in range(cols))

3. Pacific Atlantic Water Flow

  def pacific_atlantic(heights):
      rows, cols = len(heights), len(heights[0])
      pacific = set()   # reachable from Pacific
      atlantic = set()  # reachable from Atlantic

      def dfs(r, c, visited, prev_h):
          if (r,c) in visited or r<0 or r>=rows or c<0 or c>=cols:
              return
          if heights[r][c] < prev_h: return  # water flows down, not up
          visited.add((r,c))
          for dr, dc in [(0,1),(0,-1),(1,0),(-1,0)]:
              dfs(r+dr, c+dc, visited, heights[r][c])

      # Flow from Pacific (top and left borders)
      for r in range(rows): dfs(r, 0, pacific, 0)
      for c in range(cols): dfs(0, c, pacific, 0)
      # Flow from Atlantic (bottom and right borders)
      for r in range(rows): dfs(r, cols-1, atlantic, 0)
      for c in range(cols): dfs(rows-1, c, atlantic, 0)

      return [[r,c] for r in range(rows) for c in range(cols)
              if (r,c) in pacific and (r,c) in atlantic]

5. Word Ladder (Shortest Transformation)

  def ladder_length(beginWord, endWord, wordList):
      word_set = set(wordList)
      if endWord not in word_set: return 0
      queue = deque([(beginWord, 1)])
      visited = {beginWord}
      while queue:
          word, length = queue.popleft()
          for i in range(len(word)):
              for c in 'abcdefghijklmnopqrstuvwxyz':
                  new_word = word[:i] + c + word[i+1:]
                  if new_word == endWord: return length + 1
                  if new_word in word_set and new_word not in visited:
                      visited.add(new_word)
                      queue.append((new_word, length + 1))
      return 0
  # BFS on implicit graph where nodes are words and edges connect
  # words that differ by one letter. Classic BFS shortest path.

6. Surrounded Regions

  def solve(board):
      rows, cols = len(board), len(board[0])
      def dfs(r, c):
          if r<0 or r>=rows or c<0 or c>=cols or board[r][c] != 'O': return
          board[r][c] = 'S'  # safe
          for dr, dc in [(0,1),(0,-1),(1,0),(-1,0)]:
              dfs(r+dr, c+dc)

      # Mark all 'O's connected to border as safe
      for r in range(rows):
          dfs(r, 0); dfs(r, cols-1)
      for c in range(cols):
          dfs(0, c); dfs(rows-1, c)

      # Flip: 'O' → 'X', 'S' → 'O'
      for r in range(rows):
          for c in range(cols):
              if board[r][c] == 'O': board[r][c] = 'X'
              elif board[r][c] == 'S': board[r][c] = 'O'

7. Course Schedule (Cycle Detection)

  def can_finish(numCourses, prerequisites):
      graph = defaultdict(list)
      for course, pre in prerequisites:
          graph[pre].append(course)
      # 0=unvisited, 1=visiting, 2=visited
      state = [0] * numCourses
      def dfs(node):
          if state[node] == 1: return False  # cycle
          if state[node] == 2: return True   # already processed
          state[node] = 1
          for nb in graph[node]:
              if not dfs(nb): return False
          state[node] = 2
          return True
      return all(dfs(i) for i in range(numCourses))

8. Number of Connected Components in Undirected Graph

  def count_components(n, edges):
      graph = defaultdict(list)
      for u, v in edges:
          graph[u].append(v); graph[v].append(u)
      visited = set()
      def dfs(node):
          visited.add(node)
          for nb in graph[node]:
              if nb not in visited: dfs(nb)
      count = 0
      for node in range(n):
          if node not in visited:
              dfs(node); count += 1
      return count

9. Find if Path Exists in Graph

  def valid_path(n, edges, source, destination):
      if source == destination: return True
      graph = defaultdict(list)
      for u, v in edges:
          graph[u].append(v); graph[v].append(u)
      visited = {source}
      queue = deque([source])
      while queue:
          node = queue.popleft()
          for nb in graph[node]:
              if nb == destination: return True
              if nb not in visited:
                  visited.add(nb); queue.append(nb)
      return False

BFS vs DFS: When to Use Which

  Use BFS when:
  ─────────────────────────────────────────────────────────────
  - Shortest path in unweighted graph
  - Level-order processing
  - Finding closest node with some property
  - Multi-source propagation (rotting oranges, walls and gates)

  Use DFS when:
  ─────────────────────────────────────────────────────────────
  - Cycle detection
  - Topological sort
  - Finding all paths
  - Checking connectivity (components, bipartite)
  - Backtracking problems on a graph

  Both work for:
  ─────────────────────────────────────────────────────────────
  - Connected components (either visits all nodes in component)
  - Bipartite check
  - Most "reachability" questions

  Key graph complexity:
  ─────────────────────────────────────────────────────────────
  BFS/DFS traversal:   O(V + E) time, O(V) space
  Adjacency list:      O(V + E) space
  Adjacency matrix:    O(V^2) space
  V = vertices, E = edges
