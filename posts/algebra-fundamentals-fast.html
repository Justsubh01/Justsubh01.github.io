Algebra from Zero to Solid: Build Real Fundamentals Fast

Stop memorising tricks you don't understand. This guide walks you through every core idea in algebra — from what a variable actually means to quadratics and functions — with worked examples, common-mistake warnings, MIT/Harvard-calibre practice exercises (easy → challenge), and a 4-week study plan so you can move fast without skipping the foundations.

Most people learn algebra by memorising steps. Move the number to the other side. FOIL the brackets. Cross-multiply. They get through a test, forget everything in two weeks, then wonder why calculus feels impossible six months later.

This post does something different: it builds understanding first, speed second. Every rule you'll read here comes with a reason. Once you know the reason, you never have to memorise the rule — you can re-derive it on the spot. That's what "real fundamentals" means.

Each section ends with practice problems graded Easy → Medium → Hard → Challenge. The Hard and Challenge problems are drawn from competition mathematics and MIT/Harvard entrance-exam style thinking. Click "Show answer" to reveal the full worked solution.

1. The Right Mindset Before You Start

Algebra is not about numbers. It's about patterns and relationships. A number like 7 is specific. The letter x represents an entire class of possible values — and the equation tells you which specific value it must be. That shift from specific to general is the entire point.

Algebra is arithmetic made general. Everything you already know about numbers still applies — you're just working with unknowns instead of knowns.
  • Ask "why does this work?" every time you use a rule. If you can't answer, look it up before moving on.
  • Do problems by hand. Reading examples is not the same as solving them. Mathematics is a doing skill, not a watching skill.
Key insight
You are allowed to check your answer every single time. Substitute your solution back into the original equation. If both sides are equal, you're right. If not, find the step where it went wrong. This habit builds accuracy faster than anything else.

2. Number Systems — What Lives on the Number Line

  • Natural numbers (ℕ): 1, 2, 3, 4, … Counting numbers.
  • Whole numbers: 0, 1, 2, 3, …
  • Integers (ℤ): …−3, −2, −1, 0, 1, 2, 3, …
  • Rational numbers (ℚ): Any p/q where p, q are integers and q ≠ 0.
  • Irrational numbers: Cannot be written as a fraction. √2, π, e — non-repeating, non-terminating decimals.
  • Real numbers (ℝ): All of the above.

Properties of Real Numbers

  • Commutative: a + b = b + a   and   ab = ba
  • Associative: (a + b) + c = a + (b + c)
  • Distributive: a(b + c) = ab + ac
Sign rules
(positive) × (negative) = negative
(negative) × (negative) = positive
5 − (−3) = 5 + 3 = 8
Common mistake
−3² = −9   (negative is NOT squared)
(−3)² = 9   (negative IS squared, parentheses matter)

2.1Easy

Evaluate without a calculator:   −4² + (−4)²

Show answer

−4² = −(16) = −16   (the negative is NOT being squared)

(−4)² = (−4)(−4) = +16

Answer: −16 + 16 = 0

2.2Easy

Classify each number as Natural, Integer, Rational, or Irrational (choose the most specific category):   √9,   −7/3,   0.12121212…,   √7,   0

Show answer

√9 = 3 → Natural number (and also integer, rational)

−7/3 → Rational (fraction of two integers)

0.1212… = 12/99 = 4/33 → Rational (repeating decimals are always rational)

√7 → Irrational (7 is not a perfect square)

0 → Whole number / Integer

2.3Medium

Given a = −3 and b = 5, evaluate   (a − b)² − (a² − b²)   without expanding first. Use structure.

Show answer

(a − b)² = (−3 − 5)² = (−8)² = 64

a² − b² = (a+b)(a−b) = (2)(−8) = −16

Answer: 64 − (−16) = 80

The key insight: factor a² − b² as (a+b)(a−b) before substituting — it's faster and less error-prone.

2.4Medium

Prove that the product of two irrational numbers is not necessarily irrational. Give a concrete example and explain.

Show answer

Example: √2 × √2 = 2, which is rational.

Another: √2 × √8 = √16 = 4, also rational.

A statement is disproved by finding a single counterexample. The product can be irrational (e.g., √2 × √3 = √6, which is irrational), but it doesn't have to be. The claim "always irrational" is false.

2.5Hard

Let a and b be real numbers such that a + b = 3 and ab = 1. Find a&sup4; + b&sup4; without solving for a or b individually. (This is a classic MIT algebraic identity exercise.)

Show answer

Step 1. Find a² + b² using (a+b)² = a² + 2ab + b²:

9 = a² + 2(1) + b²  ⇒  a² + b² = 7

Step 2. Find a&sup4; + b&sup4; using (a² + b²)² = a&sup4; + 2(ab)² + b&sup4;:

49 = a&sup4; + 2(1)² + b&sup4;  ⇒  a&sup4; + b&sup4; = 47

Answer: 47. The pattern: power sums can always be bootstrapped from lower power sums using Newton's identities.

2.6Challenge

Prove that √2 is irrational. Your proof must be rigorous and complete (this is the classic proof taught in MIT 18.100).

Show answer

Proof by contradiction.

Assume √2 is rational. Then √2 = p/q for integers p, q with q ≠ 0, and the fraction is in lowest terms (GCD(p,q) = 1).

Squaring: 2 = p²/q², so p² = 2q². This means p² is even, so p must be even (if p were odd, p² would be odd). Write p = 2k.

Then (2k)² = 2q² ⇒ 4k² = 2q² ⇒ q² = 2k². So q² is even, meaning q is also even.

But now both p and q are even, contradicting GCD(p,q) = 1.   Contradiction.

Therefore √2 is irrational.   ■

3. Variables and Expressions

A variable is a placeholder for a number we don't know yet. An algebraic expression combines variables, numbers, and operations.

  • Term: piece of an expression separated by + or −. In 3x + 5: terms are 3x and 5.
  • Coefficient: the number multiplying the variable. In 3x: coefficient is 3.
  • Like terms: same variable to the same power. 3x and 7x are like; 3x and 7x² are not.
Simplify: 5x + 3y - 2x + 8y - 1

= (5x - 2x) + (3y + 8y) - 1
= 3x + 11y - 1
Expand: 4(2x - 3)

= 8x - 12
Common mistake
4(2x − 3) ≠ 8x − 3 — you must distribute to every term inside.
✍ Practice Exercises — Variables & Expressions
3.1Easy

Simplify:   3(2x − 4) − 2(x + 5) + 7

Show answer 
= 6x - 12 - 2x - 10 + 7
= (6x - 2x) + (-12 - 10 + 7)
= 4x - 15
3.2Medium

Without solving for p and q individually: given p + q = 5 and p − q = 3, find p² − q².

Show answer

Use the factoring identity: p² − q² = (p + q)(p − q) = 5 × 3 = 15

No need to find p = 4, q = 1 first — the identity does it directly.

3.3Medium

Simplify: (a + b)² − (a − b)². What does the result tell you geometrically?

Show answer

(a+b)² = a² + 2ab + b²

(a−b)² = a² − 2ab + b²

Difference = (a² + 2ab + b²) − (a² − 2ab + b²) = 4ab

Geometric interpretation: this is the difference in area between two squares of side (a+b) and (a−b) respectively — it equals the area of four rectangles each of dimensions a×b.

3.4Hard

Given x² + 1/x² = 7, find x³ + 1/x³. (Classic competition algebra; do not solve for x.)

Show answer

Step 1. Find x + 1/x. Use (x + 1/x)² = x² + 2 + 1/x² = 7 + 2 = 9. So x + 1/x = 3 (taking positive root for now).

Step 2. Use the identity: x³ + 1/x³ = (x + 1/x)³ − 3(x + 1/x)

= 27 − 9 = 18

If x + 1/x = −3, then x³ + 1/x³ = −27 + 9 = −18. Both ±18 are valid.

3.5Challenge

Given a/b + b/a = 1, find a³/b³ + b³/a³. (This appeared in a Harvard-MIT Math Tournament warm-up.)

Show answer

Let t = a/b + b/a = 1. Then (a/b)² + (b/a)² = t² − 2 = −1.

For cubes: (a/b)³ + (b/a)³ = (a/b + b/a)[(a/b)² − 1 + (b/a)²]

= t · [(t² − 2) − 1] = 1 · (−1 − 1) = −2

This problem looks scary but yields immediately to the sum-of-cubes identity: x³ + y³ = (x+y)(x² − xy + y²) with x = a/b, y = b/a, xy = 1.

4. Order of Operations (and Why It's Not Just a Rule)

Order of operations is a convention to eliminate ambiguity. Without it, 2 + 3 × 4 could mean 20 or 14. We all agree on one interpretation.
BODMAS / PEMDAS
Brackets • Orders (exponents) • Division & Multiplication (left to right, equal) • Addition & Subtraction (left to right, equal) 
Evaluate: 3 + 4² ÷ 2 - (1 + 3)

Step 1 Brackets:   3 + 4² ÷ 2 - 4
Step 2 Exponents:  3 + 16 ÷ 2 - 4
Step 3 Division:   3 + 8 - 4
Step 4 Add/Sub:    7
Key insight
12 ÷ 4 × 3 = 3 × 3 = 9, not 1. Division and multiplication have equal priority — resolve left to right.
✍ Practice Exercises — Order of Operations
4.1Easy

Show answer 
Brackets:   2³ - 4(4) ÷ 2 + 1
Exponents:  8 - 4(4) ÷ 2 + 1
Multiply:   8 - 16 ÷ 2 + 1
Divide:     8 - 8 + 1
Add/Sub:    1
4.2Medium

Insert exactly one pair of parentheses to make the equation true:   2 + 3 × 4 − 1 = 13

Show answer

(2 + 3) × 4 − 1 = 5 × 4 − 1 = 20 − 1 = 19 — not 13.

2 + 3 × (4 − 1) = 2 + 3 × 3 = 2 + 9 = 11 — not 13.

(2 + 3 × 4) − 1 = 14 − 1 = 13 ✓

Answer: (2 + 3 × 4) − 1 = 13. This forces the addition to happen after the multiplication, which is the default anyway — so this is a subtle trap: no parentheses were actually needed here, and the problem is testing whether you evaluate carefully.

4.3Hard

Using the digits 2, 3, 4, 5 in that order (no rearranging), and any combination of +, −, ×, ÷, and parentheses, how many distinct integer values can you produce? List at least 8 distinct values.

Show answer

Sampling of values (not exhaustive):

2 + 3 + 4 + 5 = 14
2 + 3 + 4 × 5 = 25
2 + 3 × 4 + 5 = 19
2 × 3 + 4 + 5 = 15
2 × 3 × 4 + 5 = 29
2 × 3 × 4 × 5 = 120
(2 + 3) × (4 + 5) = 45
2 × (3 + 4) × 5 = 70
(2 + 3) × 4 - 5 = 15
2 ^ 3 + 4 × 5 = 28   (if exponentiation allowed)

There are dozens of distinct values. The insight: placement of parentheses and choice of operator dramatically changes the result. This is why order of operations must be unambiguous.

5. Linear Equations — The Heart of Algebra

Whatever you do to one side of an equation, you must do to the other. The equation is a balance scale. Keep it balanced. 
Solve: 5x - 3 = 2x + 9

5x - 2x = 9 + 3
3x = 12
x = 4   ✓
Solve: x/2 + x/3 = 5

Multiply every term by LCD = 6:
3x + 2x = 30
5x = 30 → x = 6   ✓
Special cases:
2x + 3 = 2x + 7  →  3 = 7  (No solution, ∅)
2x + 3 = 2x + 3  →  3 = 3  (All real numbers)

5.3Medium

Trains problem: Two trains start 400 km apart and travel toward each other. Train A travels at 80 km/h, Train B at 70 km/h. How long until they meet, and where?

Show answer

Let t = time in hours. Combined speed = 150 km/h.

Distance covered = 150t = 400 ⇒ t = 8/3 hours ≈ 2 h 40 min

Train A covers 80 × 8/3 = 640/3 ≈ 213.3 km from its start.

5.4Hard

Find all values of k for which the equation (k − 2)x = 3k − 4 has no solution. Then find the value that gives infinitely many solutions.

Show answer

No solution: coefficient of x = 0 but RHS ≠ 0. k − 2 = 0 ⇒ k = 2. Check RHS: 3(2) − 4 = 2 ≠ 0. So k = 2 gives no solution (0 = 2).

Infinitely many: coefficient = 0 AND RHS = 0. Already k = 2 makes LHS zero; RHS = 2 ≠ 0, so there's no k that gives infinitely many here (since 3k − 4 = 0 ⇒ k = 4/3, but then k − 2 = −2/3 ≠ 0, giving a unique solution).

For any k ≠ 2: x = (3k − 4)/(k − 2). For k = 2: no solution.

5.5Challenge

A jar contains red and blue marbles. If you add 4 red marbles, one-third of the marbles are blue. If instead you add 8 blue marbles, one-half are blue. How many red and blue marbles are there originally?

Show answer

Let r = red, b = blue. Two conditions:

b / (r + b + 4) = 1/3   →   3b = r + b + 4   →   2b - r = 4     ...(1)
(b + 8) / (r + b + 8) = 1/2  →  2(b+8) = r+b+8  →  b - r = -8   ...(2)

From (1) − (2): b = 12. Substitute: 12 − r = −8 ⇒ r = 20.

Answer: 20 red, 12 blue.

Check (1): 12/(20+12+4) = 12/36 = 1/3 ✓   Check (2): 20/(20+20) = 20/40 = 1/2 ✓

6. Inequalities

The flip rule
Multiply or divide by a negative number ⇒ flip the inequality sign.
−2x > 4 ⇒ x < −2 
Solve: -2x + 3 > 7

-2x > 4
x < -2    ← flip! (divided by -2)

Solution: (-∞, -2)
Compound: -1 ≤ 2x + 3 < 7

Subtract 3:  -4 ≤ 2x < 4
Divide by 2: -2 ≤ x < 2

Solution: [-2, 2)

6.2Medium

6.3Hard

Find all real x satisfying:   x² − 5x + 6 > 0. Use a sign chart. (This technique is essential for calculus.)

Show answer

Factor: (x − 2)(x − 3) > 0. Critical points: x = 2 and x = 3.

Sign chart:
x < 2:    (−)(−) = + > 0  ✓
2 < x < 3: (+)(−) = − < 0  ✗
x > 3:    (+)(+) = + > 0  ✓

Solution: (−∞, 2) ∪ (3, ∞)

6.4Challenge

Prove the AM-GM inequality for two positive reals: for all a, b > 0,   (a + b)/2 ≥ √(ab). Equality holds when? (This is a foundational inequality in analysis.)

Show answer

Proof. Start from the fact that any real number squared is non-negative:

(√a − √b)² ≥ 0 for all a, b ≥ 0.

Expand: a − 2√(ab) + b ≥ 0 ⇒ a + b ≥ 2√(ab) ⇒ (a + b)/2 ≥ √(ab).

Equality holds if and only if √a = √b, i.e., a = b.

This proof requires no machinery beyond algebra — yet it's the cornerstone of dozens of hard competition problems and analysis theorems.

7. Systems of Equations

Substitution

y = 2x - 1  ... (1)
3x + y = 9  ... (2)

Sub (1) into (2): 3x + 2x - 1 = 9 → x = 2, y = 3

Elimination

2x + 3y = 12   ... (1)
4x - 3y = 6    ... (2)

Add: 6x = 18 → x = 3, y = 2

7.1Easy

Solve:   x + y = 7,   x − y = 3

Show answer

Add both equations: 2x = 10 ⇒ x = 5. Subtract: 2y = 4 ⇒ y = 2.

Solution: (5, 2)

7.2Medium

Solve the 3-variable system:   x + y + z = 6,   2x − y + z = 3,   x + 2y − z = 2

Show answer 
Add eq1 and eq2: 3x + 2z = 9       ...(4)
Add eq1 and eq3: 2x + 3y = 8       ...(5)
Subtract eq3 from eq2: x - 3y + 2z = 1  ...(6)

From (1): z = 6 - x - y. Sub into (4):
3x + 2(6 - x - y) = 9 → x - 2y = -3   ...(7)

From (5) and (7):
2x + 3y = 8
x - 2y = -3  → x = 2(x-2y=-3) → x = 2y-3

Sub into (5): 2(2y-3) + 3y = 8 → 7y = 14 → y = 2
x = 2(2)-3 = 1, z = 6-1-2 = 3

Solution: x = 1, y = 2, z = 3

7.3Hard

Find all real pairs (x, y) satisfying:   x² + y = 10   and   x + y² = 10. (MIT 18.01 warm-up style; exploit symmetry.)

Show answer

Symmetry observation: If (x, y) is a solution, so is (y, x). Subtract the two equations:

x² − x + y − y² = 0 ⇒ (x² − y²) − (x − y) = 0 ⇒ (x−y)(x+y−1) = 0

Case 1: x = y. Then x² + x = 10 ⇒ x² + x − 10 = 0 ⇒ x = (−1 ± √41)/2.

Case 2: x + y = 1. Then y = 1 − x. Sub into first: x² + 1 − x = 10 ⇒ x² − x − 9 = 0 ⇒ x = (1 ± √37)/2.

Total: 4 solutions (two from each case).

7.4Challenge

For what values of k does the system   2x + ky = 4,   kx + 8y = 4k   have: (a) exactly one solution, (b) infinitely many solutions, (c) no solution? (This is a classic linear algebra question that bridges algebra and matrix theory.)

Show answer

The system matrix has determinant D = 2(8) − k(k) = 16 − k².

(a) One solution: D ≠ 0 ⇒ k² ≠ 16 ⇒ k ≠ ±4

(b)/(c) When D = 0: k = 4 or k = −4.

k = 4: System becomes 2x + 4y = 4 and 4x + 8y = 16. Second = 2 × first (8 = 8 ✓). Infinitely many solutions.

k = −4: System: 2x − 4y = 4 and −4x + 8y = −16 ⇒ second = −2 × first. But 2x − 4y = 4 and −4x + 8y = −16 = −2(2x − 4y) = −8 ≠ −16. No solution.

8. Polynomials — Adding, Subtracting, Multiplying

Vocabulary
Monomial (1 term) • Binomial (2 terms) • Trinomial (3 terms)
Degree = highest exponent 
(3x² + 2x - 1) + (x² - 5x + 4) = 4x² - 3x + 3

(x + 3)(x - 2)  →  x² - 2x + 3x - 6  =  x² + x - 6
Special products
(a + b)² = a² + 2ab + b²
(a − b)² = a² − 2ab + b²
(a + b)(a − b) = a² − b²
✍ Practice Exercises — Polynomials
8.1Easy

Multiply and simplify:   (3x − 2)(2x + 5)

Show answer 
F: 3x · 2x = 6x²
O: 3x · 5  = 15x
I: -2 · 2x = -4x
L: -2 · 5  = -10

Result: 6x² + 11x - 10
8.2Medium

Find the product:   (x − 1)(x + 1)(x² + 1)(x&sup4; + 1). What pattern do you notice?

Show answer

(x − 1)(x + 1) = x² − 1

(x² − 1)(x² + 1) = x&sup4; − 1

(x&sup4; − 1)(x&sup4; + 1) = x&sup8; − 1

Pattern: each pair uses the difference of squares identity. The chain telescopes. Multiplying by (x + 1) first turns a degree-8 problem into three applications of a² − b². This trick is used in MIT problem sets to evaluate limits and sums.

8.3Hard

A polynomial P(x) has roots r, s, t. Express r² + s² + t² in terms of the coefficients of P(x) = x³ + ax² + bx + c using Vieta's formulas.

Show answer

Vieta's formulas:

r + s + t = −a

rs + rt + st = b

rst = −c

Use the identity: r² + s² + t² = (r + s + t)² − 2(rs + rt + st)

= (−a)² − 2b = a² − 2b

Example: x³ − 6x² + 11x − 6 has roots 1, 2, 3. Sum of squares = 1 + 4 + 9 = 14. Formula: (−(−6))² − 2(11) = 36 − 22 = 14 ✓

8.4Challenge

Let P(x) = x³ − 3x² + 3x − 1. Recognise the structure, factor completely, and find P(101) without a calculator. (This problem tests pattern recognition — a key MIT admissions skill.)

Show answer

Recognise: P(x) = (x − 1)³   (perfect cube pattern: (a−b)³ = a³ − 3a²b + 3ab² − b³ with a = x, b = 1)

P(101) = (101 − 1)³ = 100³ = 1,000,000

Without recognising the structure, you'd compute a 6-digit cube by hand. Pattern recognition eliminates 99% of the work.

9. Factoring — The Most Important Skill in Algebra

Always start: is there a GCF? Then count terms and apply the right method.

Factor: 2x² + 7x + 3  (AC method)

ac = 6. Find two numbers: multiply to 6, add to 7 → 1 and 6.
2x² + x + 6x + 3 = x(2x+1) + 3(2x+1) = (x+3)(2x+1)
Special patterns:
a² - b²  = (a+b)(a-b)            ← difference of squares
a³ - b³  = (a-b)(a²+ab+b²)       ← difference of cubes
a³ + b³  = (a+b)(a²-ab+b²)       ← sum of cubes
Factoring decision tree
GCF? → 2 terms (diff of squares/cubes)? → 3 terms (trinomial)? → 4 terms (grouping)? → verify by expanding.
✍ Practice Exercises — Factoring
9.1Easy

Factor completely:   (a) 4x² − 9    (b) x² − 7x + 12

Show answer

(a) Difference of squares: (2x)² − 3² = (2x + 3)(2x − 3)

(b) Find two numbers: multiply to 12, add to −7 → −3 and −4: (x − 3)(x − 4)

9.2Medium

Factor completely:   x&sup4; − 16   (hint: factor twice)

Show answer 
x⁴ - 16 = (x²)² - 4²
= (x² + 4)(x² - 4)       ← first factoring
= (x² + 4)(x + 2)(x - 2) ← factor again

Note: x² + 4 is irreducible over the reals (no real roots).

9.3Hard

Factor completely over the integers:   x&sup6; − 1

Show answer 
x⁶ - 1 = (x³)² - 1² = (x³ - 1)(x³ + 1)

x³ - 1 = (x - 1)(x² + x + 1)   ← diff of cubes
x³ + 1 = (x + 1)(x² - x + 1)   ← sum of cubes

Final: (x - 1)(x + 1)(x² + x + 1)(x² - x + 1)

Note: x² + x + 1 and x² - x + 1 have negative discriminants
so they are irreducible over the reals.
9.4Challenge

Simon's Favourite Factoring Trick: Find all pairs of positive integers (m, n) with m ≥ n such that 1/m + 1/n = 1/6. (This is a classic competition problem solved by algebraic factoring.)

Show answer

Multiply through by 6mn: 6n + 6m = mn.

Rearrange: mn − 6m − 6n = 0. Add 36 to both sides:

mn − 6m − 6n + 36 = 36 ⇒ (m − 6)(n − 6) = 36

Now find all factor pairs of 36 with m ≥ n (so m − 6 ≥ n − 6):

(m-6, n-6)  →  (m, n)
(36, 1)    →  (42, 7)
(18, 2)    →  (24, 8)
(12, 3)    →  (18, 9)
(9,  4)    →  (15, 10)
(6,  6)    →  (12, 12)

Five pairs total. Verify: 1/12 + 1/12 = 1/6 ✓, 1/7 + 1/42 = 6/42 + 1/42 = 7/42 = 1/6 ✓

9.5Challenge

Factor:   a&sup4; + 4b&sup4;   (Sophie Germain Identity — this factorisation is famous and counterintuitive because it looks like a sum of squares, which is "usually" irreducible.)

Show answer

Add and subtract 4a²b²:

a⁴ + 4b⁴ = a⁴ + 4a²b² + 4b⁴ - 4a²b²
          = (a² + 2b²)² - (2ab)²
          = (a² + 2b² + 2ab)(a² + 2b² - 2ab)

This identity is used in number theory to prove that numbers of the form n&sup4; + 4 are composite for n > 1 (e.g., 2&sup4; + 4 = 20 = 4 × 5).

10. Quadratic Equations

Form: ax² + bx + c = 0   (a ≠ 0). Three solution methods.

Factoring: x² + x - 6 = 0  →  (x+3)(x-2) = 0  →  x = -3 or 2

Completing the square: x² + 6x - 7 = 0
  x² + 6x + 9 = 16  →  (x+3)² = 16  →  x = 1 or -7
Quadratic Formula
x = (−b ± √(b² − 4ac)) / 2a

D = b² − 4ac:   D > 0 (two real),   D = 0 (one real),   D < 0 (no real)
✍ Practice Exercises — Quadratic Equations

Solve by factoring:   x² − 7x + 10 = 0

Show answer

(x − 2)(x − 5) = 0 ⇒ x = 2 or x = 5

10.2Medium

Without solving, use Vieta's formulas to find the sum and product of roots of   3x² − 7x + 2 = 0.

Show answer

For ax² + bx + c = 0: sum = −b/a, product = c/a.

Sum = 7/3,   Product = 2/3

Verify: roots are x = 2 and x = 1/3. Sum = 7/3 ✓ Product = 2/3 ✓

10.3Medium

For what values of k does x² + kx + k = 0 have exactly one real solution?

Show answer

Set discriminant = 0: k² − 4k = 0 ⇒ k(k − 4) = 0 ⇒ k = 0 or k = 4

k = 0: equation x² = 0, root x = 0. k = 4: x² + 4x + 4 = (x+2)² = 0, root x = −2.

10.4Hard

If α and β are roots of x² − 3x + 1 = 0, find α&sup5; + β&sup5; without solving for α and β explicitly. (MIT 18.01 style power-sum bootstrapping.)

Show answer

By Vieta's: α + β = 3, αβ = 1. Define S_n = α^n + β^n.

Recurrence: S_n = (α + β)S_{n-1} − (αβ)S_{n-2} = 3S_{n-1} − S_{n-2}

S₀ = 2  (always α⁰ + β⁰ = 2)
S₁ = 3  (sum of roots)
S₂ = 3(3) - 2 = 7
S₃ = 3(7) - 3 = 18
S₄ = 3(18) - 7 = 47
S₅ = 3(47) - 18 = 123

Answer: α&sup5; + β&sup5; = 123

10.5Challenge

Solve completely:   x&sup4; − 5x² + 4 = 0. How many real solutions are there? (Disguised quadratic — appeared on Harvard-MIT Math Tournament.)

Show answer

Let u = x². Then u² − 5u + 4 = 0 ⇒ (u − 1)(u − 4) = 0 ⇒ u = 1 or u = 4.

x² = 1 ⇒ x = ±1    and    x² = 4 ⇒ x = ±2

Four real solutions: x = −2, −1, 1, 2.

11.2Medium

Find the domain of   f(x) = √(4 − x²) / (x − 1)

Show answer

Conditions: 4 − x² ≥ 0 ⇒ x² ≤ 4 ⇒ −2 ≤ x ≤ 2. Also x ≠ 1.

Domain: [−2, 1) ∪ (1, 2]

11.3Hard

Find all linear functions f(x) = ax + b such that f(f(x)) = x for all x (i.e., f is its own inverse). Interpret geometrically.

Show answer

f(f(x)) = a(ax + b) + b = a²x + ab + b = x for all x.

Coefficients: a² = 1 and ab + b = 0 ⇒ b(a + 1) = 0.

Case 1: a = 1. Then b(2) = 0 ⇒ b = 0. So f(x) = x (identity).

Case 2: a = −1. Then b(0) = 0 ⇒ b can be anything. So f(x) = −x + b for any b.

Answer: f(x) = x, or f(x) = −x + b for any constant b.

Geometric meaning: f(x) = −x + b is a reflection about the line y = b/2. Reflecting twice returns to the original.

11.4Challenge

The function f satisfies f(x + y) = f(x) + f(y) for all real x, y, and f(1) = 3. Find f(n) for all positive integers n, then prove that f(r) = 3r for all rational r. (Cauchy's Functional Equation — foundational in real analysis.)

Show answer

Integers: f(n) = f(1 + 1 + … + 1) = nf(1) = 3n.

f(0) = f(0 + 0) = 2f(0) ⇒ f(0) = 0.

f(0) = f(n) + f(−n) ⇒ f(−n) = −3n. So f(n) = 3n for all integers.

Rationals: Let r = p/q. Then qf(p/q) = f(q · p/q) = f(p) = 3p. So f(p/q) = 3p/q = 3r.

Therefore f(r) = 3r for all rational r.   ■

(Over the reals, assuming continuity forces f(x) = 3x for all x, but without continuity there exist pathological solutions — a deep theorem in real analysis.)

11.5Challenge

Find all functions f: ℝ → ℝ satisfying f(xy) = xf(y) + yf(x) for all real x, y. (This is a derivation functional equation — appears in MIT 18.100 and abstract algebra.)

Show answer

Step 1. Set x = y = 0: f(0) = 2f(0) ⇒ f(0) = 0.

Step 2. Set y = 1: f(x) = xf(1) + f(x) ⇒ xf(1) = 0 for all x ⇒ f(1) = 0.

Step 3. Set x = y: f(x²) = 2xf(x).

Step 4. This is the Leibniz rule for derivatives. The continuous solutions are f(x) = cx ln|x| for a constant c (with f(0) = 0), but over the rationals, f(x) = 0 for all x is also a solution family.

The key algebraic conclusion: f(1) = 0, f(0) = 0, and f satisfies the product rule — exactly like a derivative operator.

Takeaways

  • Algebra is generalised arithmetic — every rule has a reason rooted in the properties of real numbers.
  • The balance principle (do the same to both sides) explains all equation solving.
  • Factoring is the highest-leverage skill. Practice it until it's automatic.
  • Vieta's formulas let you find sums and products of roots without solving — a competition superpower.
  • The Hard and Challenge problems here are not decoration. Working through them builds a completely different level of fluency than textbook exercises alone.
  • Check every answer. Substitute back. This habit compounds over time.
  • Four weeks, 60 minutes a day, consistent practice — that's the whole plan.
